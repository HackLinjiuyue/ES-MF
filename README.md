# ES-MF
类ES语言及其解释器

支持数据类型（弱类型）：
int
double
string
boolean
function

语法:
`logout expression`
用于输出

`if(condition){}`
用于判断

`while(condition){}`
用于循环

`function name(){}`
用于定义函数

`xx()`
用于调用函数

`var name expression`
用于定义全局变量

`let name expression`
用于定义局部变量

`x=y`
赋值

逻辑运算符：`> < = ! & |`
分别为：`大于 小于 等于 不等于 与 或`

算数运算符：`+-*/`
